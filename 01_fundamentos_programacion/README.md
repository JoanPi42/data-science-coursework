# Módulo 1: Fundamentos de Programación en Python

Este módulo contiene ejercicios introductorios de programación en Python realizados como parte de un curso de Ciencia de Datos.

El objetivo principal fue desarrollar bases sólidas de lógica de programación, sintaxis de Python y resolución de problemas mediante código.

## Temas cubiertos

* Variables y tipos de datos
* Operadores aritméticos y lógicos
* Condicionales `if`, `elif` y `else`
* Ciclos `for` y `while`
* Listas, tuplas y diccionarios
* Funciones
* Manejo básico de errores
* Ejercicios de lógica de programación

## Herramientas utilizadas

* Python
* Jupyter Notebook
* Google Colab

## Objetivo del módulo

Este módulo representa la base técnica necesaria para avanzar hacia análisis de datos, estadística aplicada y aprendizaje automático. Los ejercicios incluidos muestran el uso de Python para resolver problemas básicos, estructurar instrucciones y trabajar con datos simples.
