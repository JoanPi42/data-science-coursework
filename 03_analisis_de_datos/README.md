# Módulo 3: Introducción al Análisis de Datos

Este módulo contiene prácticas de análisis de datos con Python, enfocadas en el uso de pandas, limpieza de datos, exploración de datasets y visualización.

## Contenido

- Uso básico de pandas
- Manipulación de DataFrames
- Limpieza de datos
- Revisión de valores faltantes
- Tratamiento de duplicados
- Estadística descriptiva
- Análisis exploratorio de datos
- Visualización con Matplotlib y Seaborn
- Detección de valores atípicos
- Correlación entre variables

## Proyecto del módulo

El proyecto final utiliza el dataset Titanic de Seaborn para realizar un análisis exploratorio de datos. Se incluyen procesos de limpieza, imputación de valores faltantes, visualizaciones y conclusiones sobre los factores relacionados con la supervivencia de los pasajeros.
