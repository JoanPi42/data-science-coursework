# Módulo 2: Fundamentos Estadísticos

Este módulo contiene ejercicios y prácticas introductorias de estadística aplicada con Python.

## Contenido

- Introducción a NumPy y Matplotlib
- Estadística descriptiva
- Medidas de tendencia central
- Medidas de dispersión
- Percentiles
- Correlación
- Pruebas de hipótesis
- Regresión lineal simple
- Descenso de gradiente

## Proyecto del módulo

El proyecto final utiliza el dataset `diamonds` de Seaborn para realizar un análisis estadístico del precio de diamantes. Se incluyen visualizaciones, análisis de correlación, prueba de hipótesis y un modelo de regresión lineal simple.
