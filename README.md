# Módulo 4: Introducción al Aprendizaje Automático

Este módulo contiene prácticas introductorias de aprendizaje automático con Python.

## Contenido

- Regresión básica con TensorFlow
- Clasificación con K-Nearest Neighbors
- Evaluación de modelos de clasificación
- Matriz de confusión
- Métricas de precisión, recall y F1-score
- Clustering con K-Means
- Método del codo
- Silhouette score
- Regresión lineal simple
- Proyecto final aplicado

## Proyecto final

El proyecto final analiza el mercado de vehículos eléctricos a partir de un dataset real. Se incluyen limpieza de datos, análisis exploratorio, pruebas estadísticas, regresión, clustering y recomendaciones estratégicas.

## Datos

Los datasets utilizados se encuentran en la carpeta `data/`, siempre que su tamaño y permisos permitan compartirlos.
